Overview:
This is a simple cookbook website featuring a collection of breakfast, lunch, and dinner recipes. 
The website is designed to provide an easy way to explore a variety of recipes for each meal of the day. 
Each recipe includes a list of ingredients and instructions, along with images to guide users.

Pages:
 - Home Page (index.html)
 - Breakfast Page (breakfast.html)
 - Lunch Page (lunch.html)
 - Dinner Page (dinner.html)\

Key Features:
 - Header: Displays the website or specific page title in large text.
 - Sections: Each section contains a recipe, with an image and table for ingredients and instructions.
 - Footer: Displays a thank you message and encourages users to enjoy their meal.
